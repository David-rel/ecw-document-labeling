FAX INTAKE – eCW WORKFLOW (Delegators → Reviewed Team → TE Team)

CORE RULE

Most incoming documents REQUIRE a TE.

If unsure → Create TE → Assign to Clinical Admin → STOP.

Step 1: Delegators (Fax Inbox)

Goal: Create a TE for MOST incoming faxes straight from the Fax Inbox, assign it to Clinical Admin, and stop.

Delegators DO

  • Create TE (Encounter)
  • Assign TE to Clinical Admin
  • Update Provider and Facility
  • Use TE name: Main idea only (see TE naming below)

Delegators DO NOT

  • Label documents
  • Interpret content
  • Route to provider

Step 2: Document Reviewed Team (Relabel + Close)

Goal: Relabel documents correctly and close them. Add Document Insight notes into the TE so the TE Team can act fast.

  • Relabel document correctly
  • Close the document
  • Add key Document Insight details into the existing TE (summary, urgency, next action)

Step 3: TE Team (Clinical Admin Queue)

Goal: Review TE + Document Insight notes, take action, and assign to provider when required.

  • Review TE content + Document Insight notes
  • Route/assign to provider if provider review is required
  • Track follow-up ownership to completion

TE Creation Rules (Delegators)

Create a TE for:

  • Prior Auth (Medication / Imaging)
  • Results (Labs / Imaging)
  • Hospital (ER / Admit / Discharge)
  • Orders / Clarifications
  • Referrals needing action
  • DME / Equipment

Do NOT create a TE for:

  • Delete-only items
  • Informational consult notes
  • Immunization notices
  • Final PA approvals only
  • Duplicate faxes
  • Records with no request

TE Naming Standard (Main Idea Only)

Use only one of the following as the TE title/category:

  • Prior Auth
  • Results
  • Hospital
  • Orders
  • Referral
  • DME

GOLDEN RULE

If unsure → Create TE → Assign to Clinical Admin → STOP.
