eCW Fax Intake, Document Labeling & TE Workflow

Purpose:
Establish a single, standardized workflow for handling incoming faxes, creating Telephone Encounters (TEs), relabeling documents, and routing actionable items to Clinical Admin and providers.
Golden Rule:
If unsure → Create TE → Assign to Clinical Admin → STOP

TE Title Standards (Delegators)

  • Results
  • Imaging
  • Orders
  • Pre-Op
  • Rx
  • Medication PA
  • Imaging PA
  • DME
  • Referral
  • Hospital
  • Consult Note (action required only)

Reviewed Team: Document Insight (Required Fields)

Reviewed Team Responsibility:
Relabel and close documents, then add Document Insight into the TE so the TE Team can act without rework.

Required Document Insight Block (Paste into TE)

DOCUMENT INSIGHT (Reviewed Team)
- Fax Type:
- Key Summary (1–2 lines):
- Action Needed: YES / NO
- Who Owns Next Step: Provider / Clinic Staff / Clinical Admin / Referrals / Med PA / Imaging PA
- Due Date / Timeframe (if stated):
- Urgency: Routine / Time-Sensitive / STAT
- Provider FYI Needed: YES / NO

Field Rules

  • Fax Type must match TE title standards.
  • Key Summary is limited to 1–2 lines. No copy/paste dumping.
  • Action Needed must be explicitly marked YES or NO.
  • Due Date / Timeframe is required if mentioned anywhere in the fax.
  • Urgency must be Time-Sensitive when deadlines, pre-op dates, discharge follow-up, or peer-to-peer windows exist.
  • Provider FYI Needed should be YES only when provider review or decision is required.

Examples

DOCUMENT INSIGHT (Reviewed Team)
- Fax Type: Imaging PA
- Key Summary (1–2 lines): CT Abd/Pelvis requested. Payer requesting recent clinical notes.
- Action Needed: YES
- Who Owns Next Step: Imaging PA
- Due Date / Timeframe (if stated): 01/18
- Urgency: Time-Sensitive
- Provider FYI Needed: NO

DOCUMENT INSIGHT (Reviewed Team)
- Fax Type: Hospital
- Key Summary (1–2 lines): Discharge notification received. Follow-up recommended within 7–14 days.
- Action Needed: YES
- Who Owns Next Step: Clinical Admin
- Due Date / Timeframe (if stated): 7–14 days from discharge
- Urgency: Time-Sensitive
- Provider FYI Needed: YES

DOCUMENT INSIGHT (Reviewed Team)
- Fax Type: Consult Note
- Key Summary (1–2 lines): Ortho consult recommends PT referral and repeat imaging in 6 weeks.
- Action Needed: YES
- Who Owns Next Step: Provider
- Due Date / Timeframe (if stated): 6 weeks
- Urgency: Routine
- Provider FYI Needed: YES
Reviewed Team STOP Rule:
Once the document is relabeled, closed, and Document Insight is added to the TE, the Reviewed Team stops.
Role Summary:
Delegators route. Reviewed Team interprets. Clinical Admin executes.
